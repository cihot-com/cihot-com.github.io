数据表格

目的：大数据渲染表格会使浏览器严重卡顿。解决数据分页显示，过滤显示，排序显示，进度条显示。


环境分析：

在1个表格中，只能有1个thead和tfoot，却可以有多个tbody。


过滤功能：
A）找出符合条件的数据（加入）
B）找出不符合条件的数据（排除）
C）在目前结果基础上，追加符合条件的数据
D）在目前结果基础上，排除符合条件的数据

设计：
A）当前结果集为[]。
B）在结果集中追加符合条件的数据，并显示结果集。结果可以是空集合。记录过滤旗帜。
	结果集为
	[
		{key:'a', data:[],type:'add'}
	]
C）继续添加其他条件的数据到结果集，并显示新增数据内容。
	结果集为
	[
		{key:'a', data:[],type:'add'},
		{key:/b/, data:[],type:'add'}
	]
D) 从结果集中删除符合条件的数据，刷新显示。
	结果集为
	[
		{key:'a', data:[],type:'add'},
		{key:/b/, data:[],type:'add'},
		{key:/c/, data:[],type:'remove'}
	]
结果集类似于过滤操作的日志。
如果删除中间的操作，则可以根据日志重新演算出最新过滤结果。
如果删除最后一步操作，则有利于读取上一个操作的结果。
